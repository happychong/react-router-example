# github-notetaker应用开发 --- 代码练习

完成于 2016-07-11


## 1. start

```
$ git clone git@github.com:happychong/react-router-example.git
$ cd react-router-example
$ npm install
```
## 2. dev
### 2.1 开发调试
```
$ npm run dev
```

查看效果` http://127.0.0.1:8080`

### 2.2 代码检查
```
$ npm run lint
```

### 2.3 测试用例
```
$ npm run test:watch
```

### 2.4 产出资源
```
$ npm run build
```
